Software requirements
i)Processor:i7
ii)RAM:16GB
iii)Matlab-2022b(mandatory) or above further versions
 
Dataset Link: https://www.kaggle.com/datasets/fatmamtalaat/autistic-children-emotions-dr-fatma-m-talaats

Step 2: Extract the downloaded file
Step 3: Move all the files into the single folder

Open code folder and run only below mentioned m files.

1) Main_preprocessing.m 
2) Main_Feature_extraction.m
3) AutisticChildren_Emotions_visiual_data_classification.m(You can see the results by running this file in comment window) 

Function file
